# HumanFacePose-AI

โปรเจกต์ตัวอย่างสำหรับอัปโหลดรูปภาพ 5 มุม:
- หน้าตรง
- เงยขึ้น
- ก้มลง
- หันซ้าย
- หันขวา

แล้วระบบจะประเมินเป็นเปอร์เซ็นต์จากภาพนิ่งเท่านั้น
ไม่ใช้วิดีโอ และไม่เปิดกล้อง

## ติดตั้ง
```bash
pip install -r requirements.txt
```

## รัน
```bash
python app.py
```

เปิดเบราว์เซอร์ที่
```text
http://127.0.0.1:5000
```

## หมายเหตุ
นี่เป็น baseline สำหรับงานเริ่มต้น ใช้ face detection + heuristic scoring
ถ้าต้องการความแม่นยำสูงขึ้น สามารถเปลี่ยน analyzer.py ไปใช้ landmark model ภายหลังได้
